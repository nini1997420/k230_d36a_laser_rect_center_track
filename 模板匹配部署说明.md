# K230 1-9 数字模板匹配部署

1. 用 CanMV IDE 打开 `main_k230_digit_box_confidence_uart.py` 并运行。
2. 把电脑上的 `digit_templates` 整个文件夹复制到 K230 SD 卡根目录。
3. 部署后应存在 `/sdcard/digit_templates/digit_1_arial.pgm` 等 54 个 PGM 文件。PNG 只供电脑上预览，K230 运行时不需要。
4. 启动时终端应显示 `Digit templates loaded: 54`。

程序不加载 OCR 模型。它针对“浅色纸片上的深色数字”，在中央黄色固定区域提取暗色笔画，用行列投影自动裁出完整数字轮廓，再与 1~9 模板比较。红底白字手机画面不再作为识别目标。

LCD 右侧会显示当前数字、匹配分数 `M`、差值 `G` 和最终锁定结果。

启动时先用 5 帧学习空环境。第一个有效数字出现后，程序完整观察 4 帧，其中至少 3 帧为同一数字才确认。

终端调试行中：

- `match` 是最佳模板的 Dice 相似度，默认门槛为 `0.64`。
- `gap` 是第一名与第二名的差，默认门槛为 `0.055`。
- 调参时请以实际终端数据为准，不要同时大幅修改两个门槛。
