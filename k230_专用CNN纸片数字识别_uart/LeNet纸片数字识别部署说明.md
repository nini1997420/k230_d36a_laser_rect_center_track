# K230 纸片数字专用 CNN 识别

这是与旧 OCR/模板方案独立的新程序。

## 部署

1. 用 CanMV IDE 打开 `main_k230_digit_lenet_uart.py`。
2. 把 `digit_cnn_int8.kmodel` 复制到 `/sdcard/kmodel/digit_cnn_int8.kmodel`。
3. 不需要 OCR KModel、字典或数字模板。
4. 终端显示 `Dedicated digit CNN ready` 表示专用数字 KModel 已加载。

## 纸片要求

- 白色或浅色纸片，黑色或深色单数字。
- 数字使用较粗笔画，完整放在 LCD 黄色框中央。
- 启动的前 5 帧保持黄色框内无纸片数字。

## 判定流程

- 轮廓几何门判断数字是否存在。
- 数字保持比例缩放并居中成 32x32 输入。
- 专用 CNN 返回 0~9，类别 0 为无数字/干扰。
- 置信度低于 0.68、Top1-Top2 差值低于 0.15，或结果为 0 时拒绝。
- 完整观察 3 帧，至少 2 帧一致才锁定。

## 屏幕显示

- 摄像头、LCD、IDE 和 OSD 坐标统一为 640x480，避免 800x480 与 640x480 宽高比不同造成画面和框错位。
- 实时显示摄像头画面和黄色纸片数字识别区域。
- 找到有效数字轮廓后显示绿色轮廓框。
- 右侧显示当前 CNN 数字、置信度 P、候选差值 G 和 2/3 确认进度。
- 串口握手完成后显示 `HANDSHAKE COMPLETE`。
